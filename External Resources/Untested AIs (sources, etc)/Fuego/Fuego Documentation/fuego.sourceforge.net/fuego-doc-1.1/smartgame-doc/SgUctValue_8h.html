SgUctValue

The floating type used for mean values and counts in SgUctSearch.

The default type is double, but it is possible to use float to reduce the node size and to get some performance gains (especially on 32-bit systems). However, using float sets a practical limit on the number of simulations before the count and mean values go into "saturation". This maximum is given by 2^d-1 with d being the digits in the mantissa (=23 for IEEE 754 float's). The search will terminate when this number is reached.
