Detailed Description

Properties for nodes in a game tree.

Module SgProp defines properties that are stored in each node of a game tree. Most properties correspond to items written in the SGF file format, but there are other properties that are hidden and only used by the system.

Enumeration Type Documentation

enum SgPropPointFmt

Game dependent format for point values of SGF properties.

Enumerator:
SG_PROPPOINTFMT_GO 

Point format used in Go.

Points are written as two letters. 'aa' is top left corner.

SG_PROPPOINTFMT_HEX 

Point format used in Hex and Reversi.

Points are written as letter/number. 'a1' is top left corner. Both letters 'i' and 'j' are used ('i' is not skipped as in standard Go coordinates)
